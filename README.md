# 🎓 E-Learning Platform with WaafiPay Integration

A comprehensive Django-based e-learning platform featuring secure payment processing through WaafiPay, course management, progress tracking, and interactive quizzes.

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Django](https://img.shields.io/badge/Django-5.0-green)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-purple)
![WaafiPay](https://img.shields.io/badge/Payment-WaafiPay-orange)

## ✨ Key Features

- 🎯 Complete course management system
- 💳 Secure WaafiPay payment integration
- 📹 Video lessons with progress tracking
- 📝 Interactive quizzes and assessments
- 👥 User roles (Student, Instructor, Admin)
- 📊 Analytics and reporting
- 🔒 Secure authentication and authorization
- 📱 Responsive Bootstrap design

## 🚀 Quick Start

```bash
# Clone and setup
git clone <repo-url>
cd elearning-platform
chmod +x setup.sh
./setup.sh

# Run server
python manage.py runserver
```

Visit: http://127.0.0.1:8000

- **Backend**: Django 5.0+, Python 3.10+
- **Database**: PostgreSQL / SQLite
- **Payment**: WaafiPay API
- **Frontend**: Bootstrap 5, Font Awesome
- **Server**: Gunicorn + Nginx
